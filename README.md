Heart Disease Classification using Logistic Regression

One-paragraph description:  
This project demonstrates a basic machine learning workflow using logistic regression to classify the presence of heart disease from clinical data. It includes data loading, preprocessing, model training, evaluation, visualization, and regularization.

---

## Getting Started

These instructions will help you run the project locally for development and testing purposes.

---

### Prerequisites

You need the following software installed:

- Python 3.8+
- pip (Python package manager)

Required Python libraries:

numpy
pandas
matplotlib
scikit-learn

### Installing

Clone the repository or download the project files.

Install the required dependencies:

pip install numpy pandas matplotlib scikit-learn
